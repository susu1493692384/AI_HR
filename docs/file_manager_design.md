# 文件管理页面设计文档

## 概述
已完成文件管理页面的UI设计和实现，支持简历、附件和分析报告的统一管理。

## 主要功能

### 1. 文件分类管理
- **简历文件**：候选人上传的简历文档
- **附件材料**：作品集、证书、项目文档等补充材料
- **分析报告**：AI生成的详细分析报告

### 2. 文件操作功能
- 文件上传（支持拖拽）
- 文件预览
- 文件下载
- 批量选择和操作
- 文件删除
- 标签管理

### 3. 文件查看模式
- **网格视图**：卡片式展示，显示缩略信息和标签
- **列表视图**：表格形式，详细信息一目了然

### 4. 搜索和筛选
- 文件名搜索
- 标签筛选
- 按日期排序
- 按名称排序
- 按大小排序

## 页面布局

### 顶部区域
- 页面标题和描述
- 统计卡片（显示各类文件数量）
- 上传文件按钮

### 主内容区
- **左侧文件列表**（2/3宽度）
  - 标签页切换（简历/附件/报告）
  - 搜索和筛选工具栏
  - 文件列表（网格/列表视图）

- **右侧预览区**（1/3宽度）
  - 文件详情预览
  - 操作按钮（下载、分享）
  - 候选人信息（简历文件）

### 底部
- 无内容

## 交互设计

### 1. 文件上传流程
1. 点击"上传文件"按钮
2. 选择文件类型（简历/附件/报告）
3. 拖拽或选择文件
4. 预览待上传文件列表
5. 点击"开始上传"

### 2. 文件预览
- 点击文件卡片或列表项
- 右侧显示详细信息
- 不同文件类型显示不同预览内容

### 3. 批量操作
- 勾选多个文件
- 显示批量操作按钮
- 支持批量删除和下载

## 视觉设计

### 配色方案
- 主色调：蓝色（#3B82F6）
- 成功色：绿色（#10B981）
- 警告色：黄色（#F59E0B）
- 错误色：红色（#EF4444）

### 文件图标
- PDF：红色图标
- DOC/DOCX：蓝色图标
- ZIP：黄色图标
- 其他：灰色图标

### 状态标识
- 处理中：黄色标签
- 已完成：绿色标签
- 失败：红色标签

### 评分展示
- 80+分：绿色背景
- 60-79分：黄色背景
- <60分：红色背景

## 响应式设计
- 网格视图自适应列数（1-4列）
- 移动端优化布局
- 触摸友好的交互设计

## 技术实现

### 使用技术
- React + TypeScript
- Tailwind CSS
- 组件化架构
- 模拟数据展示

### 文件类型支持
- 简历：PDF、DOC、DOCX
- 附件：PDF、DOC、DOCX、ZIP、JPG、PNG、JPEG
- 报告：仅PDF

### 文件大小限制
- 单文件最大 10MB
- 支持多文件同时上传

## 注意事项

1. 本示例仅实现了页面UI设计，未连接真实的文件上传API
2. 所有数据均为模拟数据，用于展示交互效果
3. 文件预览功能仅UI演示，未实现真实的文件解析
4. 文件删除和下载功能仅UI演示，未实现真实操作
